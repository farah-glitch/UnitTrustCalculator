
# Unit Trust Dividend Calculator

A simple Android mobile application developed for the **Mobile Technology & Development** course assignment.

This application helps users calculate **monthly** and **total dividend returns** based on the invested amount, annual dividend rate, and number of months invested.

---

## Features

### Main Page (Calculator)

Users can input:

* Invested Fund Amount (RM)
* Annual Dividend Rate (%)
* Number of Months Invested (Max: 12)

The system calculates:

* **Monthly Dividend**
* **Total Dividend**
* Results displayed with **two decimal places**

## Formula Used

### Monthly Dividend

Monthly Dividend = (Rate / 12) × Invested Fund

### Total Dividend

Total Dividend=Monthly Dividend×Months

This formula is consistent with the assignment instructions and industry practice.
